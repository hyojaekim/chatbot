# KDU 알림이
학교 생활하면서 필요한 정보를 **카카오톡 플러스 친구**로 이용해서 학식 메뉴, 날씨, 주변 가게 정보 등을 제공해주는 서비스 입니다.

### 장점
- 간결하고 빠르게 정보를 확인할 수 있다.
- 별도의 app 설치가 필요 없다.
- 이용하기가 쉽다.

### 기술 스택
- Kotlin
- Spring Boot
- JPA

### 기능
- 학식 메뉴
- 캠퍼스 설정
- 날씨(기온, 옷차림)
- 교통 (버스나 지하철 도착 시간)
- 주로 사용하는 학교 홈페이지 이동
- 자유로운 검색 기능
- 연락처 검색 (주변 가게 또는 학교 전화번호 등을 검색 할 수 있다.)
- 건의하기
